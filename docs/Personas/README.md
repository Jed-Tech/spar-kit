# Personas

This folder is for documenting the user personas that SPAR and `spar-kit` are designed to support.

Suggested contents over time:

- core personas
- goals and frustrations
- tool preferences
- onboarding needs
- how each persona should influence roadmap and product decisions

Current examples that seem relevant from roadmap discussions:

- AI agent power user
- early invited adopter
- new-to-coding vibecoder
